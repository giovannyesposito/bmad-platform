# FlowForge AI

FlowForge AI é uma plataforma web inovadora para desenvolvimento de projetos assistido por IA. A plataforma orquestra um time de agentes especializados (Product Managers, Arquitetos, Desenvolvedores, Designers, etc.) para trabalhar em conjunto ou individualmente na construção de software.

## Características

- 🎯 **Agentes Especializados**: 19 agentes únicos divididos nos módulos Core, BMB, BMM e CIS.
- 💬 **Chat Individual**: Acione agentes específicos para tarefas focadas em suas especialidades.
- 🎉 **Party Mode**: Um ambiente colaborativo de brainstorm e discussão onde múltiplos agentes debatem soluções baseados na relevância do tema.
- 🔄 **Workflows**: 42 workflows orquestrados cobrindo todas as fases de construção de um software, da ideação aos testes e retrospectivas.

## Stack Tecnológico

- React 19
- Vite 7
- TypeScript 5.9
- CSS Puro (Design System UI-first)

## Instalação e Execução

Para rodar a plataforma localmente:

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Acesse `http://localhost:5173` no seu navegador.
